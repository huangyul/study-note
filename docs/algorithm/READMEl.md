###### 删除有序数组的重复项

注意的重点是**有序数组**，简单的思路是暴力破解，但是复杂度为O(n)，好的思路是：使用双指针，新建两个指针i，j，指针i指向数组第一个值，指针j指向第二个值，开始使用j遍历数组，如果数组i的值不等于j的值，则将i+1的值等于j，最后返回i+1

```js
/**
 * params {array} 有序数组
 * return {number}
 */
function func(nums) {
  if(nums.length == 0) return 0

  let i = 0, j = 1
  for(; j < nums.length; j ++) {
    if(nums[i] !== nums[j]) {
      i ++
      nums[i] = nums[j]
    }
  }
  return i + 1
}
```
