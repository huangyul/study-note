# OD 刷题

## 1. 字符串最后一个单词的长度

> 简单

### 描述

计算字符串最后一个单词的长度，单词以空格隔开，字符串长度小于 5000。（注：字符串末尾不以空格为结尾）

### 输入描述：

输入一行，代表要计算的字符串，非空，长度小于 5000。

### 输出描述：

输出一个整数，表示输入字符串最后一个单词的长度。


### 解题思路

将数组拆开，然后去数组最后一位字符串的长度

  while((line = await readline())) {
